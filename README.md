# Maldix YouTube Downloader

A sleek, CLI-based YouTube downloader that supports individual videos and entire channels. Built with Python, `yt-dlp`, and `rich`.

## ✨ Features
- **Channel Support:** Download every video from a channel automatically.
- **MP4 Format:** Automatically merges best video and audio into MP4.
- **Auto-Skip:** Skips unavailable, private, or deleted videos without stopping.
- **Beautiful UI:** Live progress bars and download statistics.
